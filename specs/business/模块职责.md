# 模块职责

## 根级模块

| 模块 | 文件 | 职责 | 改动注意 |
|------|------|------|----------|
| npm 脚本 | `package.json` | 开发、测试、构建、校验命令 | 新增命令需同步 `运维手册.md` |
| 本地代理服务 | `proxy-server.js` | 订阅代理、测速、mihomo、静态服务 | 新增路由需补安全测试 |
| Vite 配置 | `vite.config.js` | 前端构建 | 影响端口/构建输出时同步运维文档 |
| Docker | `Dockerfile` | 容器部署 | 改基础镜像或启动方式需同步运维文档 |

## 前端模块

| 模块 | 文件 | 职责 | 不该做的事 |
|------|------|------|------------|
| 主工作台 | `src/App.vue` | 页面结构、用户事件、模式切换、状态展示 | 不承载复杂协议解析和配置生成细节 |
| 选择托盘 | `src/components/SelectionTray.vue` | 已选节点聚合操作 | 不直接修改配置生成逻辑 |
| 引导 | `src/components/OnboardingWizard.vue` | 新手引导和示例载入 | 不维护真实业务默认值 |
| 配置健康面板 | `src/components/ConfigHealthPanel.vue` | 展示配置健康报告 | 不直接解析 YAML 业务规则 |
| 装饰组件 | `src/components/decorations/` | 视觉装饰 | 不包含业务状态 |

## Composables

| 模块 | 文件 | 职责 |
|------|------|------|
| 配置生成 | `src/composables/useConfig.js` | 落地节点解析、落地输入校验、生成中转/直连/订阅整理配置、输出 YAML |
| 订阅导入 | `src/composables/useSubscription.js` | 订阅抓取、Base64/Clash YAML/节点行解析、融合导入 |
| 节点管理 | `src/composables/useNodes.js` | 节点搜索、排序、来源/地区分组、选择、信息节点隐藏、延迟测试 |
| 节点补全 | `src/composables/useNodeParams.js` | 为导出节点补齐 Clash/mihomo 需要的默认字段 |

## Utils

| 模块 | 文件 | 职责 |
|------|------|------|
| 协议解析 | `src/utils/parsers.js` | 把节点链接解析成 Clash proxy 对象 |
| 规则解析 | `src/utils/ruleParser.js` | 按顶层逗号拆分规则，避免误拆参数内逗号 |
| 配置校验 | `src/utils/clashConfigValidator.js` | 检查代理名、策略组名、规则策略引用 |
| 节点清洗 | `src/utils/proxySanitizer.js` | 导入/导出节点字段清洗、重名处理 |
| 节点来源 | `src/utils/nodeSources.js` | 来源名称、来源前缀、来源策略组、完整来源组压缩 |
| 信息节点识别 | `src/utils/nodeMetadata.js` | 识别流量、到期、官网等非代理提示节点 |
| 规则助手 | `src/utils/ruleAssistant.js` | 规则候选、冲突检测、内置站点包 |
| 配置健康 | `src/utils/configHealth.js` | 从生成 YAML 中构建链路和问题摘要 |
| mihomo 客户端 | `src/utils/mihomoClient.js` | 调用本地 mihomo 接口，格式化错误 |
| mihomo 策略 | `src/utils/mihomoPolicies.js` | 提取可测速策略 |
| 桌面适配 | `src/utils/desktop.js` | Tauri API 封装 |
| 通用辅助 | `src/utils/helpers.js` | 时间格式、YAML 高亮等通用函数 |

## Config

| 文件 | 职责 |
|------|------|
| `src/config/defaultConfig.js` | `defaultRules`、`subscriptionDefaultRules`、`fakeIpFilter` |

默认规则中：

- `{{LANDING}}` 表示落地节点专线策略。
- `{{PROXY}}` 表示通用代理出口策略。
- 订阅整理模式不应依赖真实落地节点，`{{LANDING}}` 会被归到代理出口。

## 文档同步要求

改动以下内容时必须更新本文件：

- 新增、删除、重命名模块。
- 移动函数到新的文件。
- 改变某个模块职责。
- 新增测试分类。
