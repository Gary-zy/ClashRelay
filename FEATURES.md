# ClashRelay 功能详细说明文档 📚

ClashRelay 是一个功能强大的 **Clash Meta 链式代理配置生成器**。它通过将机场订阅节点作为“前置跳板”，中继连接到私有落地节点，实现更安全的隐私保护和更优质的网络加速。

---

## 核心技术架构：链式代理 (Relay/Chain)

链式代理是本项目的灵魂功能。
- **工作原理**：利用 Clash Meta 的 `dialer-proxy` 特性，使落地节点（Socks5）通过选定的机场节点（跳板）建立连接。
- **优势**：
  - **隐藏真实 IP**：落地服务器看到的连接来自机场 IP，而非您的真实 IP。
  - **加速私有节点**：利用机场专线/BGP 节点为原本连接不畅的落地服务器（如家用宽带、特定海外 VPS）提速。
  - **规避封禁**：有效保护私有服务器不被直接探测。

## 主要功能模块

### 1. 订阅与节点管理 (Subscription & Nodes)
- **多格式支持**：兼容 Base64（V2Ray/SS）和 YAML（Clash）格式的机场订阅链接。
- **本地代理中继**：内置 `proxy-server.js` 解决订阅站点的跨域 (CORS) 限制。
- **订阅历史记录**：支持保存最近使用的订阅链接，支持快速查询和删除。
- **节点筛选与搜索**：提供强大的关键字搜索、地区分组（港、美、日、新等）以及按延迟/名称/类型排序功能。

### 2. 节点质量检测 (Performance Testing)
- **实时延迟显示**：一键并行测试所有节点的延迟。
- **可视化标识**：
  - 🟢 **优秀 (<100ms)**
  - 🟠 **良好 (100-200ms)**
  - 🔴 **较慢 (>200ms)**
  - ❌ **超时/失败**

### 3. Socks5 落地节点配置
- **手动录入**：支持服务器、端口、用户名、密码和别名的详细配置。
- **智能链接解析**：粘贴 `socks5://` 格式链接后一键解析，自动填充表单。

### 4. 智能规则系统 (Rule System)
- **预设规则集**：支持一键导入 Loyalsoldier 的经典规则集（广告拦截、直连、代理等）。
- **自定义规则构建器**：图形化界面添加自定义规则，支持 `DOMAIN-SUFFIX`、`DOMAIN-KEYWORD` 等类型。
- **高级规则编辑**：支持手写多行 Clash 规则，高度灵活。
- **分流策略**：
  - **尊享组 (AI/Google)**：默认强制走高速落地节点。
  - **兜底组 (其他外网)**：智能负载均衡。

### 5. DNS 优化配置
- **Fake-IP 模式**：内置针对 Clash Meta 优化的 Fake-IP 设置。
- **国内外分流解析**：自定国内 (223.5.5.5) 与国外 (DoH) DNS 服务器，彻底消除 DNS 污染。

### 6. 配置生成与导出
- **实时预览**：内置带语法高亮的 YAML 代码预览区。
- **多样化导出**：
  - **复制 YAML**：一键复制完整配置。
  - **下载文件**：直接下载 `config.yaml`。
  - **一键导入链接**：生成 `clash://` 协议链接，支持一键导入桌面端或移动端客户端。
  - **扫码导入**：生成配置对应的二维码，支持手机端直接扫码（Stash / Clash for Android）。

## 交互与体验特性
- **配色美学**：现代化的深绿与墨黑拼色设计，支持半透明卡片和流体布局。
- **响应式适配**：完美适配电脑、平板及手机屏幕。
- **配置持久化**：自动保存您的节点信息和偏好设置，刷新页面不丢失数据。
- **一键清空**：支持一键重置所有配置和历史记录，保护隐私。
