# ClashRelay - Clash 链式代理配置生成器 🛠️

**在线演示地址**：[https://gary-zy.github.io/ClashRelay/](https://gary-zy.github.io/ClashRelay/)

ClashRelay 是一个基于 Web 的 Clash 配置生成工具，专为高阶网络用户设计。它能够通过“链式代理”技术，将你的机场订阅节点作为前置跳板（Relay），去连接私有的 SOCKS5 节点。这不仅能隐藏私有节点的真实入口，还能利用机场的优质线路进行加速。

## ✨ 主要功能列表

- **🔗 智能链式代理**
  - 自动解析机场订阅，提取所有节点构建“前置跳板”组。
  - 将私有 SOCKS5 节点配置为通过“前置跳板”连接。
  - 极大提升私有节点的隐蔽性和连通性。

- **📋 灵活的规则配置**
  - **美国尊享规则**：一键注入特定域名规则，强制走私有节点（适合流媒体或特定业务）。
  - **内网直连规则**：自动通过私有节点访问内网网段（192.168.x.x 等）。
  - **自定义规则**：支持手动添加域名关键词，自动生成 `DOMAIN-KEYWORD` 规则。

- **🛠️ 便捷操作体验**
  - **一键解析**：支持多种格式的订阅链接解析。
  - **可视化配置**：无需手写 YAML，通过图形界面拖拽或点选即可完成配置。
  - **导出灵活**：支持复制 Clash 订阅 URL 或直接下载 `.yaml` 配置文件。

- **🎨 现代化界面**
  - 采用 Vue 3 + Element Plus 构建。
  - 响应式设计，极简风格，操作流畅。

## 🚀 使用指南

### 第一步：输入配置信息
1. **机场订阅链接**：填入你购买的机场订阅地址（支持常见 Clash 订阅格式）。
2. **私有节点链接**：填入你的 SOCKS5 节点地址，格式如：`socks5://user:pass@1.2.3.4:1080`。

### 第二步：选择规则
- **快速添加规则**：输入域名关键词（如 `netflix`），点击添加。这些域名将通过链式代理访问。
- **预设选项**：
  - ✅ **使用美国尊享规则**：自动应用一组预设的高级路由规则。
  - ✅ **使用内网直连规则**：开启后，内网 IP 请求将通过链式代理转发回你的家庭/公司内网。

### 第三步：生成与使用
1. 点击底部的 **“生成配置”** 按钮。
2. **下载配置**：点击“下载配置文件”获取 YAML 文件。
3. **一键导入**：或者点击“复制 URL”，在 Clash 客户端（如 Clash Verge, Clash X）中通过 URL 导入配置。

## 💻 本地开发指南

如果你是开发者，想要修改或扩展本项目，请按以下步骤操作。

### 1. 环境准备
- Node.js (推荐 v18 或 v20+)
- npm 或 pnpm

### 2. 安装依赖
```bash
git clone https://github.com/Gary-zy/ClashRelay.git
cd ClashRelay
npm install
```

### 3. 启动开发环境
```bash
npm run dev
```
此命令会同时启动：
- **Web 前端**：http://localhost:5173
- **本地代理服务**：http://localhost:8787 (用于解决跨域请求问题)

### 4. 构建部署
项目已配置 GitHub Actions，推送到 `main` 分支会自动构建并部署到 GitHub Pages。

```bash
npm run build
```
